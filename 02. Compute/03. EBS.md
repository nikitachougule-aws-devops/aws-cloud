# Amazon EBS (Elastic Block Store)

- It is a block storage service for Amazon EC2.
- It provides persistent storage that can be attached to an EC2 instance like a hard disk.
- Even if you stop an EC2 instance, the data stored on EBS remains.

---

## Why Use EBS?

* Store operating systems
* Store application data
* Store databases
* Persistent storage for EC2 instances

---

## How EBS Works

```text
EC2 Instance
      │
      ▼
EBS Volume
```

- An EBS volume is attached to an EC2 instance and acts as its storage.

---

## EBS Volume

An **EBS Volume** is a virtual hard disk.

Features:

* Can be attached to an EC2 instance
* Data persists after instance stop
* Can be resized
* Supports snapshots for backup

---

## EBS Snapshot

- An **EBS Snapshot** is a backup of an EBS volume.
- Snapshots are stored in Amazon S3 (managed by AWS).

You can use a snapshot to:

* Restore lost data
* Create a new EBS volume
* Launch new EC2 instances

---

## Volume Types

| Type | Best For                                       |
| ---- | ---------------------------------------------- |
| SSD  | Operating systems, web applications, databases |
| HDD  | Large files, backups, logs                     |

---

## EBS vs Instance Store

| EBS                     | Instance Store                                   |
| ----------------------- | ------------------------------------------------ |
| Persistent storage      | Temporary storage                                |
| Data remains after stop | Data is lost if the instance stops or terminates |
| Supports snapshots      | No snapshots                                     |
| Suitable for production | Suitable for temporary data                      |

---

## Resize an EBS Volume

You can increase:

* Storage size
* Performance (IOPS)
* Throughput

without creating a new volume.

---

## Key Points

* EBS stands for **Elastic Block Store**.
* EBS provides persistent storage for EC2.
* An EBS Volume is like a virtual hard disk.
* Snapshots are used for backup and recovery.
* EBS volumes can be resized.

## Amazon S3 (Simple Storage Service)

- It is an object storage service
- Used to store and retrieve data from anywhere over the internet.
- It is highly scalable, durable, and secure.
- S3 stores data as **objects** inside **buckets**.

---

## S3 Terminology

### Bucket

- A **Bucket** is a container that stores objects.
- Example:

```text
Bucket: my-website
```

### Object

- An **Object** is a file stored in a bucket.
- Examples: photo.jpg, video.mp4, report.pdf

---

## How S3 Works

```text
S3
│
├── Bucket (my-website)
│      ├── index.html
│      ├── logo.png
│      └── style.css
```

---

## S3 Features

* Unlimited object storage
* High durability (99.999999999% - 11 nines)
* Highly available
* Secure with IAM and Bucket Policies
* Supports versioning
* Supports lifecycle management

---

## Common S3 Operations

* Create Bucket
* Upload Objects
* Download Objects
* Delete Objects
* Copy Objects
* Move Objects

---

## S3 Storage Classes

AWS offers different storage classes based on access frequency.

| Storage Class           | Best For                   |
| ----------------------- | -------------------------- |
| S3 Standard             | Frequently accessed data   |
| S3 Standard-IA          | Infrequently accessed data |
| S3 One Zone-IA          | Backup data in one AZ      |
| S3 Glacier              | Long-term archive          |
| S3 Glacier Deep Archive | Rarely accessed archive    |

> Storage Classes are covered in detail in a separate chapter.

---

## S3 Versioning

Versioning keeps multiple versions of the same object.

Benefits:

* Recover deleted files
* Restore previous versions
* Protect against accidental changes

---

## S3 Security

You can secure S3 using:

* IAM Policies
* Bucket Policies
* Access Control Lists (ACLs)
* Encryption

---

## Static Website Hosting

S3 can host static websites containing:

* HTML
* CSS
* JavaScript

It cannot run server-side code like PHP or Java.

---

## Key Points

* S3 stands for **Simple Storage Service**.
* Data is stored as **Objects** inside **Buckets**.
* Buckets can store unlimited objects.
* Supports versioning, encryption, and lifecycle management.
* Commonly used for backups, media storage, and static website hosting.
